                    STRUCTURE DE LA BASE DE DONNÉES
════════════════════════════════════════════════════════════════════════════

TABLE: menu_items
├── id (PK)
├── category
├── name
├── description
├── price
├── image_url
├── available
└── timestamps (created_at, updated_at)

TABLE: reservations
├── id (PK)
├── customer_name
├── customer_phone
├── customer_email
├── reservation_date
├── reservation_time
├── guests_count
├── special_notes
├── status
└── timestamps

TABLE: orders
├── id (PK)
├── order_number (UNIQUE)
├── customer_name
├── customer_phone
├── customer_email
├── delivery_method
├── delivery_address
├── delivery_fee
├── subtotal
├── total
├── payment_method
├── payment_status
├── order_status
├── special_notes
└── timestamps

TABLE: order_items
├── id (PK)
├── order_id (FK)
├── menu_item_id (FK)
├── quantity
└── price

TABLE: contact_messages
├── id (PK)
├── name
├── email
├── message
├── status
├── created_at
└── read_at
