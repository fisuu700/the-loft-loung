## 🔧 PERSONNALISATION FACILE

### Pour changer une photo de fond:

Dans `styles_with_photos.css`, cherchez la section et modifiez l'URL:

```css
/* Exemple: Changer le fond du Hero */
.hero {
    background-image: url('assets/images/NOUVELLE_PHOTO.jpg');
}
```

### Pour ajuster l'opacité des superpositions:

```css
/* Exemple: Hero overlay */
.hero::before {
    background: linear-gradient(
        135deg,
        rgba(47, 93, 62, 0.7),  /* Changez 0.7 (70% opacité) */
        /* ... */
    );
}
```

### Pour modifier les couleurs:

Changez les variables au début du fichier CSS:
```css
:root {
    --sage-green: #VOTRE_COULEUR;
    --primary-gold: #VOTRE_OR;
    --pink-blossom: #VOTRE_ROSE;
}
```
