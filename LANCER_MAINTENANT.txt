SI ERREUR
════════════════════════════════════════════════════════════════

Erreur "port 3000 already in use"?
  → taskkill /F /IM node.exe
  → npm start

Erreur "Cannot find module"?
  → npm install
  → npm start

Aucune réponse?
  → Exécuter: backend\fix-backend.ps1

Pas sûr?
  → Lire: BACKEND_QUICK_FIX.md
