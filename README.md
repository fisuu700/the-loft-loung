## 🎨 Personnalisation

### Modifier les Couleurs
Éditez les variables CSS dans `styles.css`:
```css
:root {
    --primary-gold: #d4af37;     /* Couleur dorée */
    --bg-dark: #1a1a1a;          /* Fond sombre */
    --text-white: #ffffff;        /* Texte blanc */
}
```

### Modifier le Numéro WhatsApp
Cherchez `wa.me/21612345678` dans `index.html` et remplacez par votre numéro.

### Modifier les Horaires
Dans `index.html`, section Contact:
```html
<span>Lundi - Dimanche: 8h00 - 23h00</span>
```

### Modifier la Position Google Maps
Dans `index.html`, cherchez `iframe` et modifiez l'URL `src`.

### Ajouter des Photos à la Galerie
1. Ajoutez vos images dans `assets/images/`
2. Dans `index.html`, section Gallery, ajoutez:
```html
<div class="gallery-item">
    <img src="assets/images/votre_image.jpg" alt="Description">
    <div class="gallery-overlay">
        <i class="fas fa-search-plus"></i>
        <p>Titre</p>
    </div>
</div>
```

### Modifier les Témoignages
Dans `index.html`, section Testimonials, modifiez le contenu des `testimonial-card`.
