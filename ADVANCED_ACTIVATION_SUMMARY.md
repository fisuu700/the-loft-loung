## 🔧 Customization Examples

### Make Animations Faster
In `styles_3d_advanced.css`:
```css
@keyframes fadeInDown {
    /* Change from 0.8s to 0.4s */
    animation: fadeInDown 0.4s ease-out;
}
```

### Change Ripple Color
In `script_3d_advanced.js`:
```javascript
background: radial-gradient(circle, rgba(212,175,55,0.8), transparent);
                                  // Change these RGB values
                                  // Example: rgba(255,100,150,0.8) for pink
```

### Disable Mouse Follow
In `index.html`, comment:
```javascript
// new MouseFollowEffect();  // Disabled
new ScrollReveal3D();
```

### Make Cards Float Less
In `styles_3d_advanced.css`:
```css
@keyframes float {
    0%, 100% { transform: translateY(0px); }
    50% { transform: translateY(-5px); }  /* was -15px */
}
```
